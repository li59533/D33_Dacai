# 交互协议
版本：V0.0.1
## 1.说明
基于“MCU私有协议规范” **版本：V0.1.18**
## 2.支持命令
1.标准CMD
```c
typedef enum
{

    MCUCmd_StdFWVersionGetReq    = 11,   // 获取版本号请求
    MCUCmd_StdFWVersionGetResp   = 12,   // 获取版本号应答

}MCUCmd_e;
```
2.私有CMD

```c
typedef enum
{
	MCUCmd_SelfUPData = 128 , // 上报数据
}MCUCmd_Self_e;
```

### 2.1 私有CMD说明
* 1.MCUCmd_SelfUPData = 128 , // 上报数据
payload格式：TLV
**Tag：**
|Tag|Len|Value|说明|
|---|---|---|---|
|0|4||D值，float类型|
|1|4||C值，float类型|
